# Date puzzle audio bits (placeholders)

Replace these placeholder files with real short MP3 clips:

- **0.mp3** – sound hint for digit 0 (used in default date 20.06.2018)
- **1.mp3** … **8.mp3** – sound hints for digits 1–8

The game maps each button’s digit to `/audio/date-bits/{digit}.mp3`.  
Until you replace the files, playback may fail silently; the UI and build paths are valid.
